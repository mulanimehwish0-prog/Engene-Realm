# 💙 Engene Hub

A fan-made community website created for ENGENEs — a cozy digital space to explore member profiles, albums, updates, and ENHYPEN's journey.

## ✨ Overview

Engene Hub is a themed web project designed to bring together information about ENHYPEN in a clean, interactive, and aesthetic interface.

The project focuses on creating a user-friendly experience with organized sections, responsive layouts, and a fan-community inspired design.

## 🌟 Features

- 🏠 Landing page with a welcoming introduction
- 👥 Member profiles section
- 💿 Discography / album timeline
- 📰 Updates section
- 🎨 Custom themed UI design
- 📱 Responsive layout
- 🧭 Navigation between different pages
- 💙 Fan-focused user experience

## 🛠️ Built With

- HTML / CSS / JavaScript
- React (if applicable)
- TypeScript (if applicable)
- Node.js (if applicable)
- AI-assisted development tools
- Git & GitHub

## 🚀 Project Goals

This project was created to explore:

- Building a complete web experience from an idea
- Designing user interfaces
- Organizing content for a community-based website
- Using modern AI tools to assist development and problem solving
